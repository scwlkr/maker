# Missing Features

These are missing from the current repo. They are not documented as implemented
elsewhere.

- Packaged release, installer, or versioned binary.
- Tagged release history.
- HTTP API server.
- Authenticated multi-user boundary.
- Service manager configuration for systemd, launchd, or another host process
  manager.
- OpenRouter model probing in the Go CLI.
- Verified deployment guide for Docker Compose or another host runtime.
- Structured project roadmap.
